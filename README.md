# NOKTA DİZAYN — ND GROUP
## Mimari Proje Portalı v5.0

### Kurulum
1. Node.js kurun (nodejs.org)
2. Terminal/Komut satırında proje klasörüne girin
3. `npm install` çalıştırın
4. `npm run dev` ile test edin
5. `npm run build` ile yayınlanacak dosyaları oluşturun
